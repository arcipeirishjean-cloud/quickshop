QUICK START GUIDE
=================

1. Navigate to the project folder:
   cd c:\websites\Quickshop\public

2. Start a local server:
   - Python 3: python -m http.server 8000

   - Node.js (http-server): http-server -c-1

3. Open browser and visit:
   http://localhost:8000
